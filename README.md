# badminton-rank
Ranking code for badminton tournaments using Microsoft's TrueSkill algorithm


## Getting Started


### Dependencies

Uses the python packages:
* TrueSkill (https://trueskill.org/)
* BeautifulSoup (https://www.crummy.com/software/BeautifulSoup/bs4/doc/)
* urllib3 
* requests
* pandas
* scipy
